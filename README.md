# amnz

- bee genome antimicrobial machine learning
- metagenome assembled from the bee genome 
- a multiclass classification approach using jumping, sparsekmer, miniizers specific for machine learning, neighbor kmer approach for machine learning.

- this can be used for any metagenome assembled and used for the training.
